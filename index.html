Mwsoft64

Wilson Lab Software

Welcome to Wilson Lab Git Hub Page.

This repository hosts the source code for software used by members of the Wilson Lab to extract, processes, and analyze electrophysiological data.

Obtaining the Source Files

Clone using git 

$ mkdir src_dir
$ cd src_dir
$ git init
$ git remote add origin https://github.com/wilsonlab/mwsoft64.git

Download an archive

Download .zip Download .tar.gz

Compiling the Software

To compile the software under Ubuntu 12.04 cd into src_dir and invoke make. When the process finishes the compiled binaries can can then be found under: src_dir/build/bin

Build Dependencies

  • libx11-dev
  • build-essential

Runtime Dependencies

  • csh
  • xfonts-75dpi
  • xfonts-100dpi
