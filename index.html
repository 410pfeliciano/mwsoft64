Mwsoft64

Wilson Lab Software

Welcome to Wilson Lab Git Hub Page.

This repository hosts the source code for software used by members of the Wilson Lab to extract, processes, and analyze electrophysiological data.

Obtaining the Source Files

Clone using git 

$ git clone https://github.com/wilsonlab/mwsoft64.git

Download an archive

Download .zip Download .tar.gz

Compiling the Software

To compile the software under Ubuntu 12.04: 

$ sudo apt-get install build-essential libx11-dev csh xfonts-75dpi xfonts-100dpi
$ cd mwsoft64
$ make all
If compiliation is successfull, units tests will be extecuted to ensure that the resulting binaries can accruately open and manipulate the various wilson lab data files.

Build Dependencies

  • libx11-dev
  • build-essential

Runtime Dependencies

  • csh
  • xfonts-75dpi
  • xfonts-100dpi
